PyBlot Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station/date

URL example for querying the temperature in a specific station ID and date: http://127.0.0.1:5000/api/v1/10/1988-10-25

URL example for all data in a specific station ID: http://127.0.0.1:5000/api/v1/10/

URL example for all data in a year taken within a specified station ID: http://127.0.0.1:5000/api/v1/yearly/10/1988
